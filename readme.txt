This plugin helps to remove URL Field from comment form in wordpress

== Description ==
The plugin allows you to disable/Remove the URL/Website field from Comment form in any wordpress theme. 

Features of our plugin:

1. Remove URL Field in any wordpress theme for free

== Installation Guide==
#### From within WordPress
1. Visit 'Plugins > Add New'
1. Search 'Remove URL Field from Comment Form in Wordpress'
1. Install and Activate the plugin.

#### Manually
1. Download and unzip the file.
1. Upload the folder in public_html/wp-content/plugins
1. Activate from the plugin page and Enjoy.

The plugins inject the code directly to functions.php to remove URL Field from the comment form
